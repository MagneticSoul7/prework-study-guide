Terminal Commands


  • ls Lists the directories(aka folders) and files contained inside of whichever directory you are currently in.
  • cd Lets you navigate forward/downward to a child directory within whichever directory you are currently in.
  • cd .. Lets you navigate backward/upward one directory (back to the parent directory).
  • mkdir Creates a new directory (aka new folder) inside of whichever directory(folder) you are currently in.
  • touch Creates a new file inside of whichever directory(folder) you are currently in.

HTML


    The HEAD & BODY:


    1. The HEAD element contains all the non-visible parts of a website (contains information about the webpage).
    2. The BODY element, however, contains all the visible parts of a website, and is divided into three parts:
      1. header (not to be confused with head)
      2. main
      3. footer

    Some helpful HTML tags to remember:

    < b > Makes text bold.
    < i > Makes text italic.
    < ol > Makes an ordered list.
    < ul > Makes an unordered list.
    < li > Makes a list item starting with a bullet point.
    < a > Makes a hyperlink.
    < img > Inserts an image.
    < h1 > through < h6 > Makes a heading (h1 having the largest text and h6 having the smallest text).
    < p > Makes a paragraph.
    < section > Defines a section in an HTML document.
    < br > Inserts a single line break.
    & nbsp Inserts a space without a line break.

CSS


    Margin VS Padding:

  • Margin indicates how much space you want around the outside of an element. (Adds space outside an element).
  • Padding indicates how much space you want around the content inside an element. (Adds space inside an element, around the content).

    Example:

    .card { width: 80%; margin: 40px auto; border: 5px solid gray; box-shadow: 5px 10px #888888; }

    Let's break down the properties and values in this rule as follows:

  • The period preceding 'card' designates this as a class selector. Any HTML element assigned to the class 'card' will gain these property assignments from the CSS.
  • The first declaration addresses the 'width' of the element at 80% of the available width.
  • The 'margin' property assigns 40px to the top and bottom areas outside the element, and it also assigns the auto value to the left and right areas outside the element.
  • The 'border' property assigns a border with a width of 5px that is solid and gray.
  • The 'box-shadow' CSS property adds shadow effects around an element's frame. You can set multiple effects separated by commas. A box shadow is described by X and Y offsets relative to the element, blur and spread radius, and color. In this example, the X offset is 5px and defines the width or extent of the shadow on the X axis (or to the right of the box); the Y offset is 10px and defines the extent on the Y axis (or just below the box). We haven’t defined the blur or spread radius here, but you’ll learn more about this in time. We’ve used an HTML hexadecimal code to define the color.
  • In this example, the coder did not use a padding property.

Git


    Clone a Remote Repository:

    1. git clone 'SSH key goes here'
      2. (Copy the 'SSH key' from the Github repo you want to clone, then type "git clone " in the terminal, then paste the SSH key and hit enter on your keyboard. This is assuming you've already set up the SSH key functionality on your computer).

    Create a New Branch:

    1. git status (Checks what branch you are currently in).
    2. git checkout -b feature/branch-name (Creates a new branch and switches to it).
      3. (3. Write some code).

    Create a Pull Request (PR):

    1. git add -A
    2. git commit -m “briefly describe the changes you made”
    3. git push origin feature/branch-name


    Go to Github to Merge the Pull Request.



    After PR has been Merged:

    1. git checkout main
    2. git pull origin main
